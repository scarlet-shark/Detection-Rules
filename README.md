# Detection-Rules
A repository for various cybersecurity detection rules from trusted vendors and security researchers.

## Sourcing
These detection rules are sourced from public blogs, reports, and from security researchers themselves.

We respect the authors of these rules by including authorship information in the metadata of each rule. 

If you or your organization are the author(s) of, or control the copyright to, a rule and would like it removed from this archive, please create an issue requesting its removal, and your removal request will be honored.
